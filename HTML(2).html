HTML Attributes

  • All HTML elements have attributes
  • Attributes give additional information about a tag
  • Attributes are defined in the start tag

Attributes

<a> defines a link. The href="" attribute defines the url

<img> defines an image. The src="" attribute defines the location/path of the image

Width & Height Attributes


if we want to define the height & width of an image we would use the height & width attributes defined in pixels or percentage or the height or width contain it's in.
