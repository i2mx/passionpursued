CSS Gradients

In CSS there are two types of gradients:
  • Linear Gradients (linear)
  • Radial Gradients (circular)
To create a gradient, you need to define at least two colors stops, the colors you want the gradient to transition between.

Linear Gradient

background-image: linear-gradient(direction, color-stop1, color-stop2, ...);


Example: We can also define a starting point and a direction.
Example: We can also use an angle instead if we wanted more precise control
Example: We can use 'repeating-linear-gradient' instead is we wanted to repeat the gradient along the page

Radial Gradient

Radial Gradient is very similar to the linear gradient except in a circular shape
background-image: linear-radial(shape size at position, color-stop1, color-stop2, ...);
