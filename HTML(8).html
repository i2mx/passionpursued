HTML Class & Id


Classes and Ids are used to define elements,
they are defined inside of the tag
and it can be used with CSS.



An element can be assigned a class like this:
<tagname class="classname">

An element can be assigned a ID like this:
<tagname id="idname">

A class or an ID can be assigned to more than one element

A class can be refrenced with:
.classname
An ID can be refrenced with:
.ID
