How can I learn coding?

To start learning coding you can read and watch online resources to learn the basics, you could also get inperson courses which is also easier to follow
I would recommend starting with HTML, which is used to make basic webpages, technically it is not a programming language. Every webpage you find is build on HTML
Then CSS which is used to make you're HTML pages
Then you should learn JavaScript which is used to make your webpages do things and is a 'real' programming language

HTML CSS and JavaScript are used for the 'front-end' of web development and is a must know for developers

Next I would reccomend Python or C# but that would really depend on what you're trying to accomplish.


A text editor is importand when coding because it has some useful features that make it easier to understand what you have written and improves your work flow.
