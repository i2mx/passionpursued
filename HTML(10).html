HTML Forms


The <input> elements defines an input
The type of input is defined as an attribute inside the tag

Here are some examples:

Type Description
<input type="text"> Displays a text input
<input type="radio"> Displays a radio button
<input type="check"> Displays a checkbox
<input type="submit"> Displays a submit button
<input type="button"> Displays a button

The <label> element can be used as a label for the inputs.
