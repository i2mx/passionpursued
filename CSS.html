What is CSS?
What is it used for?
  • CSS stands for Cascading Style Sheets
  • We use it to describe how the html looks
  • We can use it to make our webpages look alot nicer

CSS can be added to a HTML page in 3 ways

  • Inline - using the style attribute in the tag
  • Internal - using the <style>
  • External - linking to a .css file
