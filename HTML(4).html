There are two types of lists, ordered lists and unordered lists.

Ordered List:
  1. List item 1
  2. List item 2
  3. List item 3
  4. List item 4
Unordered List:
  • List item 1
  • List item 2
  • List item 3
  • List item 4

Tables

To define a table we must use the table tag
<table> </table>

We use the tr tag to define a table row
<tr> </tr>

We use the td tag to define table cell
<td> </td>

The <th> </th> is used to define a table heading





Lists

A unordered list is defined with the <ul> </ul> tags
List items are defined with the <li> </li> tags



A ordered list is defined with the <ol> </ol> tags
